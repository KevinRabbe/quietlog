# QuietLog

QuietLog is a private offline planner for real life and gaming.

It helps users manage:

- personal reminders
- shopping lists
- game events
- useful notifications

## Core Promise

Only notify the user about things they personally created.

## Philosophy

QuietLog is designed to be:

- private
- offline-first
- reliable
- quiet
- lightweight
- user-controlled

## Not Included

QuietLog intentionally does **not** include:

- cloud sync
- user accounts
- AI integration
- ads
- subscriptions
- multiplayer
- social systems

## Pricing Direction

QuietLog is planned as a low-priced one-time purchase.

Target range:

- 4,99€–6,99€

## Initial Tech Stack

- Kotlin
- Jetpack Compose
- Material 3
- Room Database
- DataStore
- WorkManager
- AlarmManager

## Development Focus

Reliability comes before features.

A simple reminder that always works is better than a smart feature that sometimes fails.
