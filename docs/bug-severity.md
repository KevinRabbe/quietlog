# QuietLog — Bug Severity

## Critical

Examples:

- reminders do not trigger
- reminder data loss
- duplicate reminder spam
- app crash on startup
- corrupted database

Critical issues should be fixed immediately.

---

## High Severity

Examples:

- snooze broken
- notification actions broken
- reboot rescheduling broken
- navigation soft locks

---

## Medium Severity

Examples:

- UI inconsistencies
- incorrect spacing
- animation glitches
- minor workflow annoyances

---

## Low Severity

Examples:

- typo
- tiny visual issues
- small cosmetic inconsistencies

---

## Main Rule

Bugs that reduce trust in reminders are the most important bugs.
