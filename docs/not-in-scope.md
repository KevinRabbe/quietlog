# QuietLog — Not In Scope

These features are intentionally excluded from QuietLog.

## Excluded Features

- No cloud sync
- No user accounts
- No AI features
- No multiplayer
- No social systems
- No ads
- No subscriptions

## Why

QuietLog should remain:

- private
- offline-first
- lightweight
- reliable
- low-noise
- sustainable long-term

## Philosophy

QuietLog is a trusted utility app.

The goal is not to maximize engagement.
The goal is to reduce notification chaos and help users manage important things quietly.
