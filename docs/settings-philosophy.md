# QuietLog — Settings Philosophy

## Goal

Settings should provide useful control without overwhelming the user.

QuietLog should remain:

- simple
- predictable
- easy to configure

---

## Core Rule

Only expose settings that create meaningful user value.

Avoid:

- endless toggles
- duplicate settings
- confusing advanced menus

---

## Main Settings Categories

### Notifications

- default notification mode
- default snooze duration
- persistent notification defaults
- shopping reminder delay

---

### Appearance

- dark mode
- light mode later
- optional accent themes later

---

### Behavior

- auto-hide completed reminders
- shopping mode behavior
- recurring reminder defaults

---

## Main Principle

The default experience should already feel good.

Most users should not need heavy configuration.
