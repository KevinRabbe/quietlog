# QuietLog — UI Notes

## UI Philosophy

QuietLog should feel:

- calm
- clean
- lightweight
- readable
- fast
- distraction-free

Avoid:

- clutter
- excessive animations
- attention-grabbing colors
- unnecessary visual noise

---

## Navigation Structure

Bottom navigation:

- Home
- Reminders
- Shopping
- Games
- Settings

Navigation should remain simple and predictable.

---

## Home Screen

The Home screen should show:

- next reminders
- active shopping lists
- today's game events
- overdue items

No unnecessary feed systems.

---

## Shopping UI

Shopping lists should support:

- large touch targets
- fast check/uncheck behavior
- minimal friction while shopping

---

## Notification Design

Notifications should be:

- short
- actionable
- clear
- non-annoying

Examples:

- Raid starts in 10 minutes
- Still items left on your shopping list

---

## Theme Direction

Default:

- dark mode first
- soft contrast
- readable typography
