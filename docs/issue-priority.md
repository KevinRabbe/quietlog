# QuietLog — Issue Priority Rules

## Highest Priority

Issues affecting:

- reminder reliability
- notification timing
- duplicate notifications
- data loss
- app crashes

These should be handled quickly.

---

## High Priority

Issues affecting:

- shopping reminder behavior
- game event reminders
- app startup
- navigation stability
- broken persistence

---

## Medium Priority

Issues affecting:

- UI polish
- layout consistency
- animations
- workflow improvements

---

## Low Priority

Issues affecting:

- cosmetic improvements
- optional visual refinements
- small convenience tweaks

---

## Main Rule

Reliability issues always matter more than visual polish.
