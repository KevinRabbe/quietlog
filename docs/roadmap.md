# QuietLog — Roadmap

## Phase 1 — Foundation

Goal:
Create the basic Android app structure.

Includes:

- Kotlin project setup
- Jetpack Compose
- Material 3
- basic navigation
- local Room database
- settings screen

---

## Phase 2 — Personal Reminders

Goal:
Build reliable personal reminders.

Includes:

- create reminder
- edit reminder
- delete reminder
- date/time picker
- repeat options
- notification modes
- done/snooze actions
- optional phone number field
- open dialer action

---

## Phase 3 — Shopping Lists

Goal:
Build fast local shopping lists.

Includes:

- create shopping list
- add items
- check/uncheck items
- clear completed items
- shopping mode
- forgot something reminder

---

## Phase 4 — Game Events

Goal:
Build manual game event planner.

Includes:

- select installed app/game
- create event
- reminder offset
- repeat options
- open game action

---

## Phase 5 — Polish & Reliability

Goal:
Make QuietLog feel stable and trustworthy.

Includes:

- notification reliability testing
- reboot handling
- UI polish
- accessibility checks
- small bug fixes
- changelog improvements
