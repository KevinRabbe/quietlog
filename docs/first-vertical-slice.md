# QuietLog — First Vertical Slice

## Goal

Build one fully working reminder flow from start to finish.

## Includes

### Create Reminder

User can:

- enter title
- select date
- select time

## Save Reminder

Reminder stored locally using Room.

## Reminder List

Reminder appears on Home screen or Reminder screen.

## Notification Scheduling

Reminder schedules Android notification correctly.

## Notification Actions

Notification supports:

- Done
- Snooze

## Reminder Completion

User marks reminder complete.

## Not Included Yet

- recurring reminders
- shopping lists
- game events
- persistent notifications
- advanced settings
- themes

## Success Criteria

QuietLog successfully:

- saves reminder
- survives app restart
- triggers notification reliably
- updates UI correctly
- feels stable
