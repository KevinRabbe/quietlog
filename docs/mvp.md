# QuietLog — MVP

## Goal

Build a fully offline Android planner for:

- personal reminders
- shopping lists
- game events

## MVP Features

### Personal Reminders

Users can create reminders with:

- title
- date
- time
- optional repeat
- notification mode

Optional:
- phone number
- open dialer action

---

### Shopping Lists

Users can:

- create shopping lists
- add items
- mark items done
- clear completed items

Includes:
- Forgot Something Reminder

---

### Game Event Planner

Users can:

- select installed app/game
- create event title
- set event time
- set reminder before event
- open linked app from notification

---

### Notification Modes

- Off
- In-App Only
- Device Notification
- Persistent Notification

---

## Not Included In MVP

- cloud sync
- accounts
- AI
- ads
- subscriptions
- multiplayer
- social systems
- reward systems
- movement tracking
