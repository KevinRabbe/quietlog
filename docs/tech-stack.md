## Dependency Philosophy

Only add dependencies that clearly improve the project enough to justify:

- APK size increase
- maintenance cost
- security risk
- long-term complexity

Avoid:

- analytics SDKs
- ad SDKs
- tracking frameworks
- AI SDKs
