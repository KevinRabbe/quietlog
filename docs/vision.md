# QuietLog — Vision

QuietLog is a private offline planner for real life and gaming.

It helps users manage:

- personal reminders
- shopping lists
- game events
- important notifications

QuietLog is not designed to fight for attention.
It is designed to protect attention.

## Core Promise

Only notify the user about things they personally created.

## Product Identity

QuietLog should feel:

- calm
- reliable
- fast
- private
- simple
- user-controlled

## What Makes QuietLog Different

Most apps use notifications to pull users back in.

QuietLog uses notifications only when the user explicitly asks for them.

## Core Rules

- no ads
- no cloud
- no accounts
- no AI integration
- no subscriptions
- no social systems
- offline-first

## Target User

QuietLog is for people who want one trusted place for useful reminders instead of many noisy apps.

Especially useful for users who:

- disable most app notifications
- want control over reminders
- play games with timed events
- want shopping lists without spam
- prefer private offline tools

## Long-Term Goal

QuietLog should become a small, trusted Android utility that simply works.

Not bloated.
Not addictive.
Not noisy.
Just useful.
